# Go Binance.com datas reader

*Description: This library for readout history trade datas and orders from https://www.binance.ceo


## Installation


```shell
go get -u github.com/jojopoper/CoinReader/binance/...
```

## Usage

Since the platform does not provide the API interface, the current program is obtained from the platform web page, and there may be problems unresolved.

rdweb can get order and history datas, and rdweb1 can only get order information.

```go
cd exsample
go build main.go
main
```
